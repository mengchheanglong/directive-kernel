# UI Library Reference Packs

These packages were previously registered as callable capabilities but returned hardcoded reference data. They are retained as non-callable reference material.

- **shadcn/ui** — Framework-agnostic React component library (116K GitHub stars). Reference: https://ui.shadcn.com/
- **daisyUI** — Tailwind CSS component library with themeable semantic classes. Reference: https://daisyui.com/
- **React Bits** — Animated interactive React components. Reference: https://reactbits.dev/
- **React Bootstrap** — Bootstrap 5 components for React. Reference: https://react-bootstrap.netlify.app/
- **Tremor** — React chart and dashboard component system. Reference: https://tremor.so/
